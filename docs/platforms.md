# Platforms

Now above all this infrastructure sit the platforms people think are the internet.

Huge digital metropolises like:

- YouTube
- TikTok
- Meta Platforms
- Discord

In our animated world, these would appear as giant floating cities built on top of the Substrate landscape.

Billions of people visit them every day.

Watching videos.
Talking with friends.
Sharing ideas.
Building communities.

But these cities only exist because the Substrate beneath them is constantly working.
